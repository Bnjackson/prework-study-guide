Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for Hyper Text Markup Language
  • HTML is used to build a websites structure
  • A HTML document starts with a document type declaration
  • A HTML document is made up of a series of elements
  • Elements are defined by tags <>. Elements are case sensitive
  • Some elements are semantic(they give meaning to the content they contain)

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS stands for Cascading Style Sheets
  • CSS is used to add styling to a web page
  • CSS can select elements, classes or id's
  • CSS can be added inline to apply style to a single element
  • CSS can be used internally to add styles to a single HTML page
  • CSS can be used externally to add styles to multiple HTML pages. This is the industry standard.
