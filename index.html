HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML stands for Hyper Text Markup Language
  • HTML is used to build a websites structure
  • A HTML document starts with a document type declaration
  • A HTML document is made up of a series of elements
  • Elements are defined by tags <>. Elements are case sensitive
  • Some elements are semantic(they give meaning to the content they contain)

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS stands for Cascading Style Sheets
  • CSS is used to add styling to a web page
  • CSS can select elements, classes or id's
  • CSS can be added inline to apply style to a single element
  • CSS can be used internally to add styles to a single HTML page
  • CSS can be used externally to add styles to multiple HTML pages. This is the industry standard.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Git is the most widely used version control software
  • Git allows for collaboration between developers and is an integral part of agile software development
  • The repository tracks all the changes made to the files in a git project
  • A commit is a snapshot of your project at a given time
  • Git branches allow you to work on different versions of your project at the same time
  • Git allows you to merge different branches toegther

JavaScript

  • A variable is a named container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • A function is a set of statements that perform a task and must be called
  • Loops can be used to iterate over a block of code
  • An array is a type of variable that can contain more than one value
  • In JavaScript comments are created using /* */ or //
  • .console.log prints a message to the console log
  • Conditional statements are used to perform different actions based on different conditions
